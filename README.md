# 박주현 QA Engineer Portfolio

박주현 QA 경력 PDF, QA Chapter Tool, TC Maker 2.0 자료를 취합해 만든 정적 포트폴리오입니다.

## 구성

- `index.html`: 포트폴리오 본문
- `styles.css`: 반응형 스타일
- `assets/qa-chapter-tool.png`: QA Chapter Tool 공개 화면 캡처
- `assets/tc-maker-preview.png`: TC Maker 2.0 입력 모달 미리보기
- `source-notes.md`: 사용한 자료와 반영 기준

## 배포

이 폴더의 파일을 별도 GitHub Pages 저장소 루트에 업로드하면 정적 사이트로 바로 배포할 수 있습니다.

개인 전화번호는 공개용 포트폴리오 본문에 넣지 않았고, 이메일만 연락 수단으로 사용했습니다.
